# Lagos In Yankee '26 — Official Event Website

The official website for **Lagos In Yankee (LIY) '26**, Maryland's premier Afro-cultural pop-up experience. Built as a single-page site showcasing the event, its vendors, experience highlights, gallery, and ticketing/sign-up CTAs.

---

## Pages & Sections

| Section | Description |
|---|---|
| **Hero** | Full-screen video hero with event date, location, and CTAs |
| **Highlight Video** | Embedded recap from a previous edition |
| **About** | Story and mission behind LIY |
| **Experience** | What to expect — fashion, food, wellness, art |
| **Featured Vendors** | Cards for the 6 headline vendors with images and descriptions |
| **Also Joining Us** | Pill-style list of additional confirmed vendors with contact info |
| **Details** | Event date, time, and venue info |
| **Gallery** | Photo grid from past editions |
| **FAQ** | Frequently asked questions accordion |
| **Tickets** | Free attendance CTA |
| **Become a Vendor** | Vendor application sign-up section |
| **Footer** | Links, social handles, and legal |

## Tech Stack

- **HTML5** — single `index.html` page
- **CSS3** — custom properties, grid, flexbox, transitions (`styles.css`)
- **Vanilla JS** — scroll animations, mobile nav, accordion
- No frameworks or build tools — pure static site

---

## Notable Details

- Hero video compressed from 48MB → 801KB for fast load
- Scroll-triggered `fade-up` animations on all major sections
- Fully responsive — mobile, tablet, and desktop breakpoints
- Vendor pills in the "Also Joining Us" section have a gold fill hover toggle
